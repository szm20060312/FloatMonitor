# 开发需求文档

## 项目名称
cpu_mem_tool（正式名称待定）

## 项目目标
开发一款 macOS 原生系统监控应用，在菜单栏中实时显示 CPU 和内存占用率，提供弹出式详细监控面板和桌面窗口两种查看模式。

## 核心功能

### 1. 菜单栏显示
- 在 Mac 菜单栏中实时展示 CPU 占用率和内存占用率
- 支持三种显示模式（可在设置中切换）：
  - 纯文本模式：`CPU 23% MEM 45%`
  - 仅图标模式：小条形图/仪表盘图标
  - 图标+数值模式：图标配百分比数字
- 占用率数值随系统负载实时更新

### 2. 弹出面板（Popover）
- 点击菜单栏项弹出详细监控面板
- 面板内容包含：
  - **CPU**：整体使用率 + 各核心柱状图
  - **内存**：已用/总量进度条 + 压力级别
  - **网络**：下载/上传实时速率
  - **温度**：CPU/GPU 温度数值
  - **GPU**：使用率百分比
- 点击面板外部自动关闭

### 3. 桌面窗口模式
- 提供独立桌面窗口，展示完整监控数据
- 复用弹出面板的数据视图组件
- 支持窗口置顶（Float on Top）
- 窗口可拖拽、调整大小

### 4. 偏好设置
- 刷新间隔：0.5s / 1s / 2s / 5s
- 菜单栏显示模式：纯文本 / 仅图标 / 文本+图标
- 温度单位：°C / °F
- 网络速率单位：MB/s / Mbps
- 开机自动启动
- 窗口置顶开关

### 5. 历史图表
- 记录最近 60 分钟的监控数据
- 使用折线图展示 CPU/内存/网络的趋势变化
- 在弹出面板和桌面窗口中均可查看

### 6. UI 风格
- 符合 macOS 26 "液态玻璃"设计语言
- 使用 `.glassBackgroundEffect()` 半透明背景
- SF Symbols 6 图标系统
- 适配浅色/深色模式
- 平滑的数值变化动画

## 非功能需求
- 监控服务本身的 CPU 占用应低于 2%
- 应用启动后立即开始监控
- 设置变更即时生效，重启后保留

## 用户故事
1. 作为 Mac 用户，我希望在菜单栏一眼看到 CPU 和内存状况
2. 作为开发者，我想点击菜单栏查看详细的系统资源使用情况
3. 作为高级用户，我希望看到温度、网络、GPU 等更多维度的数据
4. 作为普通用户，我希望能根据自己喜好调整显示方式和刷新频率
