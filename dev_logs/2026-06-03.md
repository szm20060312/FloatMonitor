### 关键技术决策
- 使用 **SPM（Swift Package Manager）** 而非传统 .xcodeproj（跨 CLI/Xcode 兼容）
- 使用 `NSStatusBar` + `NSPopover`（AppKit 桥接）而非 `MenuBarExtra`（后者不支持富内容 Popover）
- 使用 `SystemMonitorService.shared` 全局单例模式，AppDelegate 通过 Combine `$stats.sink` 订阅更新
- App 入口使用 `@NSApplicationDelegateAdaptor` 集成 AppDelegate

### 架构说明
```
cpu_mem_toolApp (@main)
  ├── NSApplicationDelegateAdaptor → AppDelegate
  │     ├── NSStatusBar ← 订阅 $stats 更新文本
  │     └── NSPopover ← MenuBarView (SwiftUI)
  └── WindowGroup → ContentView (SwiftUI)
        └── TabContentView (概览/CPU/内存/网络)

共享数据源: SystemMonitorService.shared (ObservableObject)
```
